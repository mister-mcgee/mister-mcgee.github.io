## Installing Visual Studio Code

In yonder days of old, software developers relied on a variety of tools to write, test, build, and package software. Developers then necessarily had to learn the various quirks of each tool and regularly swap between them. Yuck! Thankfully, most programming today occurs in an IDE or *Integrated Development Environment*. 

An IDE is a tool intended to assist in software development, by bundling the functionality of several tools into one. Most modern IDEs include a code editor with features like syntax and error highlighting, code completion, testing and debugging, and building executables.

For the purposes of this tutorial we will be using VSCode also known as *Visual Studio Code* ; however, other popular choices for programming Python include IDEs like PyCharm or IDLE.

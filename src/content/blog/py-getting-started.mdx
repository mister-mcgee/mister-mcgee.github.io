# Overview

### What is Python?

If you have never written a program before, the following sentence might seem like nonsense.

> **Python** is a dynamically-weak typed, interpreted **Programming Language**.

### What is a Programming Language?

You may have heard that computers speak natively in **Binary** or **Machine Language** which is unfortunate for the majority of humans who speak **Human Language**. Fortunately for those same humans, some other very clever people have written translators that allow us to speak to machines using what often amounts to broken-english. This arcane speech is known as a **Programming Language** and these translators come in a handful of flavors, namely - **assemblers**, **compilers**, and **interpreters**.

### What is an Interpreter?

As mentioned earlier, **Python** is an **interpreted** language, meaning that it uses an **interpreter** to translate your program into **Machine Language**.

## Installing a Python Interpreter

### <a className="no-underline hover:underline" href="https://apps.microsoft.com/detail/9NCVDN91XZQP?hl=en-us&gl=US" target="_blank">Using the Microsoft Store <ArrowSquareOut className="inline"/></a>

1. Open the Microsoft Store 
    - Open the **Start Menu** by selecting the **Windows** in ( <WindowsLogo className="inline"/> ) on your taskbar or by pressing the **Windows** key <kbd> <WindowsLogo className="inline"/> </kbd> on your keyboard.
    - Begin typing "Store" to search for the **Microsoft Store**.
    - Select the **Microsoft Store** from the list of available options.
2. Install a Python Interpreter
    - Type "Python" in the search bar and select the magnifying glass ( <MagnifyingGlass className="inline"/> ) or press the **Return** key <kbd><KeyReturn className="inline"/></kbd> on your keyboard.
    - Select the *most recent* version of **Python** from the list of available options (At the time of writing it is currently <pre className="inline p-0">Python 3.12</pre>)
    - Depending on your version of **Windows** you will either see an option to <kbd>**Get**</kbd> or <kbd>**Install**</kbd> the software. Select either option to install the **Python** interpreter.

### <a className="no-underline hover:underline" href="https://www.python.org/downloads/" target="_blank">Using an Installer <ArrowSquareOut className="inline"/></a>

1. Download the Installer
    - Using your web browser, navigate to <a href="https://python.org/downloads/" target="_blank">python.org/downloads</a> and select <nobr><kbd>**Download Python 3.XX**</kbd></nobr>. Navigate the following prompt(s) and make a note of the directory where you saved the installer.

2. Run the Installer
    - Using **File Explorer** navigate to the directory where you saved the installer and double-click on the executable to run it.
    - When prompted check that the following options match.
        - *Uncheck this if you are **not** an administrator*.<br/><Square className="inline"/> <pre className="inline p-0">Use admin priveleges when installing py.exe</pre>
        - <CheckSquare className="inline"/> <pre className="inline p-0">Add python.exe to PATH</pre>
    - Select <kbd>**<ArrowRight className="inline"/> Install Now**</kbd> and leave the defaults unchanged for any following prompts.

## Installing Visual Studio Code

In yonder days of old, software developers relied on a variety of tools to write, test, build, and package software. Developers then necessarily had to learn the various quirks of each tool and regularly swap between them. Yuck! Thankfully, most programming today occurs in an IDE or *Integrated Development Environment*. 

An IDE is a tool intended to assist in software development by bundling the functionality of several tools into one. Most modern IDEs include a code editor with features like syntax and error highlighting, code completion, testing and debugging, as well as building executables.

For the purposes of this tutorial our IDE of choice will be *Visual Studio Code* or *VSCode* for short.
